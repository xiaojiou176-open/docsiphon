# Docsiphon Roadmap

This page is the repo-local roadmap entrypoint for Docsiphon.

Use it when you want the current public next-step themes without depending on
specific GitHub Discussion or Issue numbers that may change across repository
rebuilds.

## Current Themes

- expand docs-site compatibility coverage for more real-world documentation
  stacks
- strengthen the public example profiles so first-time users can move from
  README to a successful export faster
- improve the review surfaces around `manifest.jsonl`, `report.json`,
  `report.html`, and `toc.md`
